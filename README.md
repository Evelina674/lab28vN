# Лабораторна робота №28

## Тема
Серіалізація об'єктів у JSON

## Мета
Навчитися серіалізувати та десеріалізувати об'єкти у форматі JSON

## Опис
Реалізовано консольний додаток для роботи із задачами та проєктами.
Дані зберігаються у JSON-файл.

## Реалізовано
- Класи TaskItem, ProjectItem
- Репозиторій TaskRepository
- Методи:
  - Add()
  - GetAll()
  - GetById()
  - SaveToFileAsync()
  - LoadFromFileAsync()

## Демонстрація
У Main:
- створюються об'єкти
- зберігаються у JSON
- зчитуються з файлу
- виводяться у консоль
